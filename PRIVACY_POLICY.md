# Privacy Policy

**Extension:** Toggle Debug Mode  
**Effective date:** February 18, 2026

## Summary
This extension does not collect, store, transmit, sell, or share personal data.

## Data Collection
- No personal information is collected.
- No browsing history is collected.
- No analytics or tracking data is collected.
- No data is sent to external servers.

## How Permissions Are Used
- `tabs`: Used only to read and update the current tab URL when toggling configured query parameters.
- `storage`: Used only to save user-defined extension settings (query parameter rules) in browser storage.
- `offscreen`: Used only to detect light/dark theme preference for icon contrast.

## Data Sharing
No user data is shared with third parties.

## Remote Code
The extension does not download or execute remote code.

## Changes to This Policy
If this policy changes, updates will be posted in this file.

## Contact
For privacy questions, contact the extension publisher through the Chrome Web Store listing or the repository issue tracker.
